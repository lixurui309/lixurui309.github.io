　　Floodlight控制器是由Big Switch Network公司主导的基于Java语言开发的SDN控制器。Floodlight提供了功能强大的REST接口，为SDN的设计和管理提供了很大的便捷。这篇博客把之前做的关于访问REST API的实验整理出来，方便以后源代码的解读工作和增加floodlight模块的工作。
